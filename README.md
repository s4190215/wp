## Project Description
PetConnect is a static web application for pet adoption. It is built using HTML, CSS, JavaScript, and Bootstrap 5.

## Features
- Responsive design using Bootstrap
- Home page with carousel and pet cards
- Browse pets page with table
- Gallery with modal view and filtering
- Add pet form with validation and image preview
